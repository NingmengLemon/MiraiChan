

# Mirai 酱

~~3rd-Remake 版~~

此框架仿照 pycqBot 框架编写，实现了它的一部分功能，但非常简陋就是了  
（所以是自用的）

目前所有插件都已移植完毕，现在正在修 Bug 和搓新插件  
完成后会尝试部署在我的服务器上
